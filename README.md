
# Customer Churn Analysis
-Exploratory Data Analysis on a telecom customer churn dataset using Python, Pandas, and Seaborn.

## What I found
- 26.54% of customers churned
- New customers churn the fastest (first 2-3 months)
- Month-to-month contract users leave more than yearly ones
- Customers without support services churn significantly more

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib
- Jupyter Notebook
